# Sistema de Gerenciamento – Loja Mundo de Estilo

Este projeto implementa um banco de dados completo para o gerenciamento de uma loja de roupas que vende produtos masculinos, femininos e infantis, com vendas presenciais e online.

O sistema gerencia:
- Clientes
- Produtos
- Categorias
- Fornecedores
- Vendas
- Itens das vendas
- Pagamentos

---

Arquivos do Projeto

create_tables.sql
Contém todas as tabelas com:
- Chaves primárias
- Chaves estrangeiras
- Relacionamentos 1:N e N:N

insert_data.sql
Contém dados fictícios inseridos para testes:
- Clientes
- Produtos
- Categorias
- Fornecedores
- Vendas
- Pagamentos

queries.sql
Inclui:
- SELECT simples
- SELECT com WHERE
- SELECT com ORDER BY e LIMIT
- JOINs entre tabelas
- 3 comandos UPDATE
- 3 comandos DELETE
